resilience

Composable resilience for Go. Retry, backoff, circuit breaker. One primitive: func(ctx, call) error. Zero deps.

Docs → 🤖

autosolve

Self-hosted daemon that polls GitHub and dispatches AI agents. Two-phase scheduler, outbox pattern, full OpenTelemetry.

Docs → ⚙️

ghset

Declarative GitHub repository settings. Describe an existing repo into YAML, spin up a new repo from that YAML. One command.

Docs → 📦

gover

Composable governance CLI for Go projects. Version determination, release notes, cross-compilation, pre-publish staging. The missing cargo-release for Go.

Docs → 🔨

op

Transport-agnostic, language-agnostic application-layer protocol for describing operations. Solves the Expression Problem with traits.
